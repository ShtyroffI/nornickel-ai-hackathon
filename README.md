# Научный клубок — карта знаний R&D для горно-металлургической отрасли

Решение задачи хакатона: единая онтологическая база знаний и интеллектуальная система поиска для управления R&D-данными в горно-металлургической отрасли.

## 🌟 Особенности решения
- **Извлечение данных (NLP Pipeline):** Автоматический парсинг из `.pdf`, `.docx`, `.md`, `.txt`. Поддержка извлечения сущностей (материалы, процессы) и параметров (числовые значения, концентрации) с помощью YandexGPT/RuBERT.
- **Граф Знаний (Neo4j):** Построение онтологии "Материал -> Процесс -> Оборудование -> Свойство". Версионирование фактов и связей.
- **Умный семантический поиск:** Гибридный поиск по базе знаний, с поддержкой сложных многопараметрических фильтров.
- **Генерация литературных обзоров:** Автоматическое составление обзоров с разделением на "консенсус" и "противоречия", а также строгой детерминированной группировкой подтверждающих источников.
- **Сравнение технологий:** Построение таблиц сравнения параметров различных процессов (например, автоклавное vs кучное выщелачивание).
- **Выявление пробелов:** Анализ "белых пятен" в базе знаний (какие пересечения материалов, процессов и условий еще не изучены).
- **Рекомендательная система:** Подбор смежных технологий, похожих кейсов и экспертов в отрасли по запросу.
- **Защищенный доступ:** Ролевая модель и авторизация (JWT).

## 🚀 Быстрый старт

### Требования
- Docker
- Docker Compose

### Установка и запуск
1. Клонируйте репозиторий:
   ```bash
   git clone <repo-url>
   cd nornickel-ai-hackathon
   ```
2. Скопируйте файл конфигурации и впишите ключи (YandexGPT API):
   ```bash
   cp .env.example .env
   ```
3. Запустите все сервисы в Docker:
   ```bash
   docker compose up -d --build
   ```
4. Откройте интерфейс пользователя в браузере по адресу:
   ```
   http://localhost:8501
   ```
5. Логин по умолчанию:
   - **Email:** admin@example.com
   - **Пароль:** admin

## 🛠 Технологический стек
- **Backend:** FastAPI (Python 3.12)
- **Базы данных:** PostgreSQL (реляционные данные пользователей), Neo4j (граф знаний)
- **Frontend:** Streamlit, PyVis (визуализация графов)
- **AI / NLP:** YandexGPT, Langchain, spaCy, PyMuPDF, python-docx

## 📁 Структура проекта
- `app/` - Исходный код бэкенда (API, сервисы, NLP, интеграция с Neo4j)
- `ui/` - Исходный код фронтенда (Streamlit)
- `data/` - Примеры тестовых данных для загрузки
- `scripts/` - Утилиты для инициализации

Разработано для хакатона.
